##(x)ubuntu
**dark theme**

    sudo add-apt-repository ppa:noobslab/themes
    sudo apt-get update
    sudo apt-get install delorean-dark

>**Note:** perfect is delorean-dark-theme-G-3.9

**terminal colors**

> foreground: #AAAAAA or rbb = {170,170,170}
> background: #232323 or rgb = {35,35,35}
